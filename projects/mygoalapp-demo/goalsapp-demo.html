MyGoals App Demo

This is a demo version of a personal goals app and agent workflow that I built (with AI obviously). It features daily goals and a monthly calendar that highlights days in different colors, based on the number of goals completed. The personal/production version I use, has an ai agent to intelligently update goals, send reminders, and analyze goal trends, based on natural language input from my phone or computer.
