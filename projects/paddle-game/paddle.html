Paddle Ball Game

A simple implementation of the classic Paddle Ball game, built 100% by AI, using HTML5 Canvas and vanilla JavaScript. You can control the paddle with your mouse, arrow keys, or touching the screen on mobile.

Key Features & Learnings:

  • Dynamic ball movement and physics.
  • Paddle control via mouse, touch, and keyboard input.
  • Collision detection between ball, paddle, and walls.
  • Score tracking.
